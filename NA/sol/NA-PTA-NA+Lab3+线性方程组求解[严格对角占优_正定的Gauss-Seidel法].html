There is No Free Lunch

Format of function:

void Price( int n, double p[] );

where int n satisfies that 2< n ≤10000; double p[ ] is passed into the function as an array of prices p​i​​ and then will be returned storing the costs of the dishes.

Sample program of judge:

#include <stdio.h>
#define Max_size 10000 /* max number of dishes */

void Price( int n, double p[] );

int main()
{
    int n, i;
    double p[Max_size];

    scanf("%d", &n);
    for (i=0; i<n; i++) 
        scanf("%lf", &p[i]);
    Price(n, p);
    for (i=0; i<n; i++)
        printf("%.2f ", p[i]);
    printf("\n");

    return 0;
}

/* Your function will be put here */

Sample Input:

12 23.64 17.39 12.77 16.62 10.67 14.85 12.68 26.90 28.30 15.59 37.99 23.18

Sample Output:

9.20 5.58 3.24 7.00 1.99 6.36 2.25 10.01 11.52 0.50 17.65 4.88

本题的矩阵A：

这可能是NA lab里最简单的一道，只要知道Gauss-Seidel方法求线性方程组。当系数矩阵 A严格对角占优或正定时，Gauss-Seidel迭代法必收敛。
其中，x初始值是随便估计的，非常随意，它会自己收敛到最终结果，只是迭代次数不同罢了。下面程序中初始值全部设为0.


在本题中，可以简化如下：

My answer:

void Price( int n, double p[] )
{
	int i, cnt = 0;
	double x[Max_size]={0};
	while((cnt++)<200){
		for(i=0;i<n;i++){
			x[i] = ( p[i] - ( x[(i-1+n)%n]/2.0 + x[(i+1)%n]/2.0 ) ) / 2.0;
		}
	}
	
	for(i=0;i<n;i++){
		p[i]=x[i];
	}
}
