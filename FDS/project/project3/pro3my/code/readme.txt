此程序判定输入的序列是否是一个Dijsktra Sequence

在code文件夹中可找到 源代码lab3.c  macos可执行文件lab3mac 压缩包lab3.zip（解压后为lab3.exe）
运行方法：
1.1 可通过vscode打开lab3.c,点击run code 运行即可（或用自己的方式编译并运行）。
1.2 首先输入一个Graph 格式为：第一行输入Graph的vertice数m和edge数n，接下来输入n条边，格式为v1，v2 weight。(均为int类型）
1.3 在下一行给出案例数 
1.4 对于每个案例，输入一个对应输入Graph的遍历sequence
1.5 程序会给出输入是否为Dijsktra Sequence 的判断，如果是，则会输出'Yes',如果不是，则会输出‘No’。


2.如果是mac用户，可通过在terminal中输入./lab3mac的命令或直接双击来运行可执行文件lab3mac（需要为可执行文件授予执行权限，即在终端中cd进入lab3mac所在目录，输出chmod+x lab3mac命令）

3.如果是windows用户，由于pta无法直接上传.exe文件，需解压lab3.zip文件，得到lab3.exe运行，在终端中使用./lab3运行即可（同样需要如2中所述，使用chmod +x lab3.exe命令赋予执行权限。
