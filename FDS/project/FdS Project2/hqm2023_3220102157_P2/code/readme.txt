此程序判定输入的二叉树是否为红黑树
在code文件夹中可找到 源代码hqm2023_3220102157_P2.c macos可执行文件treemac 压缩包tree.zip（解压后为tree.exe）
运行方法：
1.1可通过vscode打开hqm2023_3220102157_P2.c,点击run code 运行即可（或用自己的方式编译并运行）。
1.2在第一行给出一个正整数 K（≤30），它表示总案例数。
1.3对于每个案例，第一行给出一个正整数 N（≤30），表示二叉树中的总节点数。
1.4第二行给出树的先序遍历序列，使用负号来表示红色节点。一行中的所有数字都用空格分隔。
1.5程序会给出输入树是否为红黑树的判断，如果是，则会输出'Yes',如果不是，则会输出‘No’。
1.6 在源代码tree.c 的 line 121 是可视化输入二叉树的代码（已经被注释了），如果希望看到输入二叉树的结构，可以去掉注释，重新编译运行。

2.如果是mac用户，可通过在terminal中输入./treemac的命令或直接双击来运行可执行文件treemac（需要为可执行文件授予执行权限，即在终端中cd进入treemac所在目录，输出chmod+x treemac命令）

3.如果是windows用户，由于pta无法直接上传.exe文件，需解压tree.zip文件，得到tree.exe运行，在终端中使用./tree 运行即可（同样需要如2中所述，使用chmod +x tree.exe命令赋予执行权限。
